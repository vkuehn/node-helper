# node-helper
a single javascript file to make the repeating node js stuff a one liner. 
No outside dependencies will be here

node-helper.js can be found in lib directory.

it provides these functions

  copyFileOnce
  getFileList
  getLocalIPs
  getTimeStamp
  getUserHome
  getValidInteger
  getValidString
  isDirSync
  isEmpty
  isINT
  isRunning
  isWindows
  loadFile
  log
  mkDirOnce
  runCommand
  runScript
  saveFile
  sleep
  showData
  state
